Wingin't It

Game Creation Experience

"Wingin’ It" is an arcade-style game where the user plays as Buzz to collect different objects. The player will keep collecting objects to increase their high score, being careful not to miss 3 items in a row or collect UGA, which will result in game over. Each object falls into one of three categories: common, rare, and harmful. Common items are worth 50 points, rare items are with 500, and harmful items either decrease the point value or end the game. "Wingin’ It" is meant to be a game that can be played for short periods of time.

The creation of this game allowed my group and I to combine all of the knowledge we've obtained over the course of this semester. Utilizing HTML, JavaScript, and Adobe Fresco, we created a game that could be played with ease by any user. We also had the opportunity to consider how we would continue to develop the game, if we were given more time. In which case, I would want to create a punishmnent, other than Game Over, to hitting a negative object.
