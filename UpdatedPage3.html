TwitterBot Experience

The purpose of the TwitterBot project was to manipulate the twitter interface by creating a bot that could interact with the Twitter space and function based on those inputs. A few things that the TwitterBot was tasked to do was retweet tweets made by other people, based on an inputted hashtag. As shown by the image on the lower left, our TwitterBot retweeted based on the "#music" tag. Our bot was also created to respond to the tweets with the "#music" tag and comment "This is so cool." An example of this function is shown by the image on the upper left. The last function that we implemented was created using the Wordnik API. This function would tweet rhyming words based on an inputted word.

Each of these functions were created using JavaScript. This assignment allowed me to get more comfortable with the syntax and functions of the language.
